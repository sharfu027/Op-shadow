# Database DDL Schemas

This directory holds DDL (Data Definition Language) SQL specifications organized by domain boundary.

## Contents
- **Table DDL Statements**: Clean SQL DDL for table creation, constraints, foreign keys, and indexes.
- **Domain Boundaries**: Segregated by ERP module schemas (`iam`, `master`, `procurement`, `warehouse`, `inventory`, `sfa`, `o2c`, `returns`, `finance`, `workflow`, `hrms`, `crm`, `logistics`, `reports`, `admin`, `bi`).
