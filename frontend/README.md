# Enterprise FMCG ERP - Frontend Client

Production-Grade FMCG Enterprise Resource Planning (ERP) Frontend Client built with React 19, TypeScript, Vite, Tailwind CSS, and React Router.

## Quick Start (Frontend)

```bash
# Navigate into frontend directory
cd frontend

# Install dependencies
npm install

# Run Vite dev server
npm run dev

# Run TypeScript check
npx tsc --noEmit

# Build production bundle
npm run build
```

## Architecture Summary
Follows a Feature-Based Modular Architecture with centralized services, typed REST contracts, dynamic security policy resolution, lazy-loaded modules, and shared UI components.
